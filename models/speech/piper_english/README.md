# Piper English Voicepack

Download the Piper TTS model files and place them here.

## Required Files

1. **en_US-lessac-medium.onnx** - Main model file
2. **en_US-lessac-medium.onnx.json** - Model configuration

## Download Links

Visit: https://github.com/rhasspy/piper/releases/tag/v1.2.0

Look for: `voice-en_US-lessac-medium.tar.gz`

Extract the `.onnx` and `.onnx.json` files to this folder.

## Alternative Voices

You can use any Piper voice by:
1. Downloading a different voice model
2. Updating the `model_path` and `config_path` in `config.json`

Available voices: https://github.com/rhasspy/piper/blob/master/VOICES.md
